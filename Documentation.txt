Документация по лабораторному проекту

1 Компиляция (интерпретация) проекта
	Проект написан на некомпилируемом языке python и запускается в интерпретаторе
	Проект зависит от сторонней библиотеки OpenCV for python и Qt for python, 
		Запуск под ОC Windows и Unix-подобные ОС осуществляется с помощью команды "python wtfexample.py [параметры командной строки]"
		Возможные параметры командной строки:
		--Camera	: Включает вывод исходного потока видео на экран
		--Gray		: Включает вывод переведенного в оттенки серого видеопотока
		--Binary	: Включает вывод переведенного в бинарный режим (только черные и белые пиксели) видеопотока
		--Sobel		: Включает вывод преобразованного с помощью фильтра Собеля видеопотока
		--Blur		: Включает вывод видеопотока с антишумовым фильтром
		--Diff		: Включает вывод видеопотока с фильтром поиска границ
		--Contours	: Включает вывод видеопотока с фильтром поиска замкнутых контуров
	По умолчанию проект запускается в режиме --Contours

2 Очистка проекта
	Проект во время работы не создает временных файлов, поэтому очистки не требуется.

3 Функциональность проекта
	Приложение позволяет захватывать видео с веб-камеры, установленной на компьютере, используя библиотеку OpenCV. 
	Кроме того, поддерживаются несколько режимов вывода видео на экран:
		Без дополнительных преобразований
		В режиме оттенков серого
		В бинарном (черно-белом) режиме
		С применением фильтра Собеля
		С применением антишумового фильтра
		С применением фильтра поиска границ
		С применением фильтра поиска контуров

4 Структура проекта
Приложение не экспортирует никаких интерфейсов для использования его сторонними приложениями, соответственно документировать нечего :).
